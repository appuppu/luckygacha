あなたの今日の運勢を占うシンプルなガチャアプリ

プライバシーポリシー

本プライバシーポリシーは、「今日の運勢ガチャ」（以下、「本アプリ」）において、ユーザーの個人情報の取り扱いについて定めるものです。

1. 収集する情報

本アプリは、以下の情報を収集する場合があります：

  • 広告識別子（IDFA）: 広告の配信と効果測定のため
  • デバイス情報: デバイスの種類、OSバージョン、アプリバージョン
  • 利用状況データ: アプリの使用頻度、クラッシュレポート

本アプリは、ユーザーの氏名、メールアドレス、電話番号などの個人を特定できる情報は収集しません。

2. 情報の利用目的

収集した情報は以下の目的で利用します：

  • 適切な広告の配信
  • アプリの機能改善とバグ修正
  • 利用状況の分析とサービス向上

3. 広告について

本アプリは、Google AdMob（Google LLC）を使用して広告を配信しています。

  • Google AdMobは、ユーザーの興味に基づいた広告を表示するためにCookieや広告識別子を使用する場合があります
  • Google AdMobのプライバシーポリシー: https://policies.google.com/privacy
  • 広告のパーソナライズを無効にする方法: デバイスの設定 → プライバシー → 広告から設定できます

4. 第三者への情報提供

本アプリは、以下の場合を除き、収集した情報を第三者に提供することはありません：

  • ユーザーの同意がある場合
  • 法令に基づく開示請求がある場合
  • 本アプリのサービス提供に必要な範囲で、Google AdMobなどの広告配信事業者と共有する場合

5. データの保護

本アプリは、収集した情報を適切に管理し、不正アクセス、紛失、破損、改ざん、漏洩などを防ぐための合理的な安全対策を講じています。

6. 子供のプライバシー

本アプリは、13歳未満の子供から意図的に個人情報を収集することはありません。13歳未満の方は、保護者の同意を得てから本アプリをご利用ください。

7. プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて変更される場合があります。変更がある場合は、本ページにて公開し、重大な変更の場合はアプリ内でお知らせします。
